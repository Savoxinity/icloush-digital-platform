## 一、项目背景与结论摘要

环洗朵在 iCloush 统一数字平台中的角色，并不是一个与 LAB 平行重复的“第二个品牌展示页”，而是整个多站点体系中**距离商业转化最近的专业解决方案品牌站点**。在平台方法论里，环洗朵被定义为“工业级专业信赖感”的承接者，负责把专业化学品能力、行业解决方案与销售线索收敛到可执行的 B2B 转化路径中。[2] [3]

根据品牌手册，环洗朵的底层品牌资产已经比较清晰：其叙事基点不是消费级洁净体验，而是以**30 年行业经验团队、军规级洁净技术、环保可持续配方、CMA 与 CNAS 认证实验室能力**为核心证据，向酒店、餐饮与居住空间提供三大场景的系统性解决方案。[1] 这意味着环洗朵的视觉语言不应走 LAB 的“深渊感、神秘感、未来黑域”路径，而应走**透明、洁净、理性、精密、可验证**的工业科技路径。

本次深化方案据此提出一个明确结论：**环洗朵的 VI 不应强调“神秘科技”，而应强调“可被看见的清洁科技”；不应强调情绪压迫，而应强调证据透明；不应强调暗黑奢华，而应强调军规洁净与生态理性。**

> “环洗朵诞生于上海黄浦的产品实验和研发中心，由 30 年行业经验化工新老专家团队历时 3 年研发。我们突破性地将军规级洁净技术与环保可持续配方融合……通过诸项专利技术和 CMA&CNAS 认证实验室技术，为酒店、餐厅和家庭提供真正高效的清洁解决方案。” [1]

## 二、品牌战略基线

环洗朵当前最重要的任务，不是建立抽象的品牌审美，而是让访客在极短时间内理解三个问题：**你是谁、你解决什么、为什么值得信任**。因此，本次 VI 深化以“解决方案证明型品牌”为结构原点，而不是传统快消品牌的感性包装逻辑。

| 维度 | 环洗朵应有状态 | 不应采用的状态 |
| --- | --- | --- |
| 品牌本质 | 清洁科技解决方案品牌 | 情绪化消费品牌 |
| 第一感知 | 军规级洁净、实验室可信、理性透明 | 神秘、晦暗、难以理解 |
| 商业目标 | 拉近咨询、申领样板、导购到采购入口 | 仅停留在形象展示 |
| 内容组织 | 方案、证据、参数、TDS、资质、流程 | 过度抒情、抽象概念堆叠 |
| 视觉关键词 | 洁净、锐利、秩序、显微、酶解、制图 | 厚重、黑金、侵略性、装饰过载 |

从平台整体分工来看，LAB 负责建立技术高度与品牌张力，环洗朵则负责把专业能力翻译成行业可采购、可验证、可试样、可咨询的业务语言。[2] [3] 因此，环洗朵的视觉系统必须天然适配官网、PDP、TDS 下载、样板申领与企业微信咨询等转化动作，而不仅是适配宣传画册。

## 三、品牌核心命题

### 3.1 品牌定义

环洗朵的品牌定义建议固化为：**极度透明、军规级洁净、生态可持续的次时代清洁解决方案品牌。**

这个定义由三层能力构成。第一层是“极度透明”，代表配方逻辑、工艺机制、参数指标、适用场景与交付流程都要尽量可视、可读、可下载。第二层是“军规级洁净”，代表洁净效率、系统稳定性、配料精准性与执行标准化。第三层是“生态可持续”，代表无磷、生物降解、循环包装、浓缩配方与智能分配等可持续承诺。[1]

### 3.2 品牌关键词

| 类型 | 关键词 |
| --- | --- |
| 价值关键词 | 透明、可信、稳健、洁净、节制、可持续 |
| 技术关键词 | 活性酶、自动分配、软水净化、显微结构、标准工艺 |
| 情绪关键词 | 清醒、呼吸感、秩序感、冷静、专业 |
| 业务关键词 | 解决方案、试样、技术规格、采购桥接、行业适配 |

### 3.3 品牌口号建议

在保留手册中“次时代清洁解决方案”表达的同时，建议将官网主传播口号稳定为：**次时代清洁解决方案**。这个口号已经出现在品牌手册中，兼顾科技感与商业清晰度。[1]

可与之配套的副标题建议采用：**以军规级洁净技术与可持续配方，重构酒店、后厨与居住空间的专业清洁标准。** 该副标题更适合首页首屏、方案页和企业介绍页使用。

## 四、色彩系统

本次 VI 采用你已明确确认的 **6:3:1 色彩矩阵**，并将其从“配色偏好”升级为**品牌信息分层规则**。环洗朵的色彩不是装饰用途，而是承担“洁净底层—专业支撑—活性提示”的信息结构。

| 层级 | 色值 | 比例 | 建议命名 | 主要职责 |
| --- | --- | --- | --- | --- |
| 主底色 | `#F8FAFC` | 60% | 实验室白 | 作为大面积背景、页面留白、数据模块底层、表单与技术文档底色 |
| 主识别色 | `#0047AB` | 30% | 科技深蓝 | 作为品牌标题、主导航、核心 CTA、线框与模块骨架色 |
| 活性强调色 | `#10B981` | 10% | 净化翠绿 | 作为状态点缀、酶解动效、可持续标签、交互反馈与数据高亮 |

为保证系统可扩展性，建议在数字界面中进一步扩展出一组辅助色阶，用于 hover、边框、数据图表与弱化文本处理。

| Token | 色值 | 用途 |
| --- | --- | --- |
| `hx-white-50` | `#F8FAFC` | 页面主背景 |
| `hx-white-100` | `#EEF3F8` | 卡片底色、表格奇偶行 |
| `hx-blue-700` | `#0047AB` | 一级标题、主按钮、主图形 |
| `hx-blue-900` | `#002B73` | 深色标题、hover、深层结构线 |
| `hx-green-500` | `#10B981` | 高亮标签、成功反馈、酶解轨迹 |
| `hx-green-100` | `#D1FAE5` | 绿色浅底徽章、可持续说明背景 |
| `hx-gray-500` | `#64748B` | 辅助文字、参数说明 |
| `hx-gray-900` | `#0F172A` | 主体正文、深层信息 |

这组配色的关键，不是把绿色做成第二主色，而是确保绿色只在**活性、环保、结果确认与微观流动**场景中出现。只有这样，视觉上才能保持“军规洁净”的节制感，而不会滑向日化快消的活泼廉价感。

## 五、字体系统

环洗朵的字体策略应服务于“工业可信 + 中文可读 + 参数清晰”三重目标。主标题需要几何化、具有科技秩序感；中文正文需要高可读性与长文稳定性；技术参数与编号信息则需要等宽系统来强化专业证据感。

| 层级 | 字体 | 用途 | 风格要求 |
| --- | --- | --- | --- |
| 英文/数字主标题 | Lexend 或 Inter Bold | 首页大标题、模块标题、产品英文编号、解决方案标题 | 几何感强、结构利落、字腔开阔 |
| 中文正文 | 思源黑体 | 正文说明、产品介绍、表单、段落与下载说明 | 高可读、现代、中性、不花哨 |
| 参数/技术数据 | JetBrains Mono | pH 值、稀释比、规格、SKU、实验数据、系统状态文本 | 等宽、理性、工程感 |

建议建立如下字阶逻辑。

| 文本层级 | 建议规格 | 场景 |
| --- | --- | --- |
| Hero H1 | 56–88px / 700 | 首屏主文案“次时代清洁解决方案” |
| Page H1 | 36–48px / 700 | 页面标题、方案标题 |
| Section H2 | 24–32px / 700 | 模块标题 |
| Body L | 18px / 400–500 | 重要正文、导语 |
| Body M | 16px / 400–500 | 常规正文 |
| Meta / Caption | 12–14px / 500 | 参数说明、说明文字、按钮补充信息 |
| Data Mono | 12–16px / 500 | 技术参数、实验数据、标签编号 |

字体的视觉效果应体现“有呼吸感的理性”，因此不建议出现 LAB 式超压迫字重堆叠，也不建议使用过于圆润的商务字体。标题可以大，但必须干净；正文可以密，但必须可读；参数可以多，但必须秩序化。

## 六、视觉语言原则

### 6.1 总体视觉命题

环洗朵的视觉语言应被定义为：**工业制图风格矢量纹章 + 活性酶分解污垢的科学美学 + 电子显微镜级材质证据**。

这意味着页面上的“科技感”不来自黑底霓虹，而来自以下三类视觉构成：第一，结构化线框、标注线、角标、截面式图解与精密网格；第二，液体、泡沫、纤维、污渍与酶解过程的显微视觉；第三，实验室器皿、自动分配器、工业料桶与系统设备的标准化陈列。[1]

### 6.2 图形系统

| 图形元素 | 推荐做法 | 目的 |
| --- | --- | --- |
| 工业制图线框 | 使用细线、坐标、注释线、编号角标 | 建立工程感与参数感 |
| 矢量纹章 | 以分子结构、蜂窝、液滴剖面、纤维网状结构为原型 | 构建品牌识别而非装饰噪音 |
| 数据刻度 | 在局部模块加入 pH、温度、浓度、配比等视觉刻度 | 让“洁净科技”被看见 |
| 净化轨迹 | 用翠绿细粒子或流线表达酶解与分配路径 | 承担 10% 活性高亮 |
| 微观纹理 | 用显微镜织物纤维、液滴扩散、结晶形态做底纹 | 增加科学证据氛围 |

这些元素必须遵循一个原则：**所有装饰都要像功能性信息，而不是纯装饰。** 只有这样，环洗朵才会呈现出“可信的专业洁净”气质。

### 6.3 摄影与影像风格

根据你的明确要求与品牌手册中的产品体系，环洗朵的摄影风格应围绕“材料、反应、设备、结果”四类主题展开。[1]

| 类别 | 推荐主体 | 画面要求 |
| --- | --- | --- |
| 反应类 | 化料溶解瞬间、液体扩散、泡沫酶解、污渍剥离 | 高速摄影感、冷静背景、动作清晰 |
| 显微类 | 织物纤维前后对比、污渍附着与分解、液滴微观结构 | 高倍显微、局部放大、纹理真实 |
| 设备类 | 自动分配器、蠕动泵、软水净化装置、工业桶装产品 | 正视角、标准陈列、工程品质感 |
| 场景类 | 酒店布草、后厨油污、客房清洁、操作人员标准作业 | 干净、明亮、避免廉价摆拍 |

视频与动效同样遵循这一逻辑。首页 Hero 的“活性酶分解污垢”视频背景，不应做成泛科幻 CG，而应更接近**科学实验可视化**：可见液体扩散、纤维表面变化、污渍结构被解构的过程，并用蓝白基调承载，绿色仅作为活性提示出现。

## 七、版式与界面构成

环洗朵官网的版式不应使用 LAB 的巨物压迫结构，而应建立一种**高留白、高秩序、高可读、高信任**的 B2B 版面。页面的结构感要强，但压迫感要弱；信息量可以大，但层级必须清楚。

| 版式维度 | 建议 |
| --- | --- |
| 页面骨架 | 大留白 + 中轴或偏左栅格 + 模块化分区 |
| 卡片样式 | 直角或极小圆角，弱阴影，强调细边框与线性结构 |
| 分区节奏 | “视觉证据区—文字说明区—参数/CTA 区”三段式最稳妥 |
| 边框语言 | 细线、折角、注释编号优先，避免粗重装饰边框 |
| 图文比例 | 首页以图像/影像 55%、文字 45% 为宜；PDP 以信息 60%、视觉 40% 为宜 |

建议统一三种官网核心模块样式。第一种是“技术陈述模块”，用于说明活性酶、自动分配、环保配方与测试逻辑。第二种是“解决方案模块”，用于行业场景、问题、产品组合与结果收益。第三种是“证据下载模块”，用于 TDS、检测报告、资质与样板申请。

## 八、品牌语调与文案规范

环洗朵的语言风格必须与视觉系统一致，即**专业，但不僵硬；可信，但不古板；有呼吸感，但不轻浮。**

| 维度 | 建议口径 | 不建议口径 |
| --- | --- | --- |
| 自我介绍 | 我们提供系统性的专业清洁解决方案 | 我们是最强、最好、最领先 |
| 产品表达 | 强调适用场景、机制、参数、结果 | 夸张情绪词、直播口播式形容 |
| CTA | 申请试样、下载 TDS、咨询方案、接入采购 | 立即抢购、爆款疯抢、限时秒杀 |
| 信任表达 | 引用资质、实验室、行业经验、方案体系 | 空泛口号堆叠 |
| 环保表达 | 无磷、生物降解、循环包装、智能分配 | 绿色生活式消费号召 |

建议官网统一使用以下三类 CTA。

| CTA 类型 | 中文 | 英文辅助 |
| --- | --- | --- |
| 主转化 | 申请试样 | REQUEST SAMPLE |
| 技术转化 | 下载技术规格书 | DOWNLOAD TDS |
| 商务转化 | 获取行业方案 | GET SOLUTION |

其中“申请试样”应成为全站最高频的主动作，因为这最符合环洗朵在 B2B 场景中的低门槛转化逻辑。

## 九、与 iCloush LAB. 的差异化边界

为了避免多品牌前台在统一底座上出现认知重叠，环洗朵必须在视觉、语调、内容结构和转化路径上与 LAB 明确分离。[2] [3]

| 维度 | 环洗朵 | iCloush LAB. |
| --- | --- | --- |
| 品牌角色 | 解决方案与转化承接 | 品牌实验场与技术张力输出 |
| 色彩气质 | 白 + 深蓝 + 绿色，透明洁净 | 黑 + 金属 + 低占比高光，深渊神秘 |
| 视觉逻辑 | 显微、制图、参数、工程秩序 | 巨物、黑域、神性纹章、压迫性构图 |
| 用户动作 | 申请试样、下载 TDS、咨询方案、扫码采购 | 建立认知、沉浸浏览、品牌记忆 |
| 文案方式 | 专业说明、参数证据、行业适配 | 高张力命名、世界观式表达 |
| 页面节奏 | 清晰导览、低理解门槛 | 强叙事、强氛围、较高感知门槛 |

这张差异表非常关键。它不是对 LAB 的否定，而是在统一平台中为每个品牌保留清晰角色。环洗朵若复制 LAB 的暗黑史诗感，会直接削弱其作为 B2B 商业承接站点的效率。

## 十、数字界面落地建议

环洗朵的官网、PDP、下载页与后台配置项，建议围绕以下数字界面规则统一执行。

| 场景 | 建议规范 |
| --- | --- |
| 首页 Hero | 白底或浅冷灰底，深蓝标题，科学可视化视频为主视觉，绿色仅作反应轨迹 |
| 解决方案页 | 按“场景痛点—方案组合—流程机制—效果收益—CTA”组织 |
| 产品详情页 | 左侧产品信息与卖点，右侧固定样板申请/采购桥接浮窗 |
| TDS 下载 | 使用统一的蓝白技术文档卡片，突出版本号、适用产品、更新时间 |
| 表单系统 | 大字号标签、浅底输入框、深蓝按钮、绿色成功反馈 |
| 企业微信/二维码区 | 不做广告式堆叠，采用“技术顾问联络 / B2B 采购入口”双卡结构 |

建议后续若进入前端开发阶段，可直接将这套 VI 转译为 CSS Token、组件变量与内容模块模板，从而保证环洗朵前台与后台内容配置在执行上保持一致。

## 十一、执行优先级建议

如果后续马上进入官网视觉设计与前端开发阶段，建议优先锁定四件事。第一，锁定首页 Hero 的视频基调与三张关键显微图风格，因为这是整个品牌气质的第一入口。第二，锁定“申请试样 / 下载 TDS / 扫码采购”三类转化组件的统一样式，因为这些模块会在首页、解决方案页与 PDP 高频复用。第三，锁定产品矩阵卡片与技术规格表格的版式，因为它直接决定 B2B 浏览效率。第四，锁定与 LAB 的主题隔离边界，避免在实现阶段再次回流到暗黑视觉。

## 十二、结论

环洗朵的 VI 深化，不是简单地把现有官网换成更“科技”的蓝色风格，而是要把品牌手册中已经存在的三项能力——**军规级洁净、系统性方案、生态可持续**——转译为一套可被官网、PDP、TDS、样板申领与采购桥接统一调用的视觉语言。[1] [2] [3]

因此，本方案最终建议将环洗朵塑造成一个**看起来像实验室、说话像工程团队、转化路径像成熟 B2B 解决方案站点**的品牌系统。它与 LAB 共享平台底座，但必须在感知上形成鲜明区隔：LAB 是“黑域品牌张力”，环洗朵是“可视化洁净科技”。这条边界一旦立住，后续官网设计、产品包装、技术文档与线下展陈都会更容易形成统一风格。
